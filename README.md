# Worldow Rank 

### A Javascript project using NextJs and API REST

Using it you can search for information about all the countries (not really all)

### infos:

    Name
    Population
    Area (km²)
    Gini coefficient: "represents the income inequality"
    Subregion
    Language (most used)
    Currency
    Native name
    Neighbouring countries
